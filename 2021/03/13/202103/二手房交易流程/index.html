二手房交易流程

Posted on 2021-03-13 In 生活

二手房交易流程图

购房资质

  1. 北京户口
  2. 北京工作居住证
  3. 五年连续社保或纳税
  4. 是否首套贷款（认房又认贷）
  5. 首套首付35% 二套首付60%
    首付35%仅指普通北京住宅。
    认房又认贷，是以家庭为单位判断的
  • 北京普通住宅
    1. 容积率1.0以上
    2. 建筑面积140以下
    3. 总价：五环内468万以下，五环外374万以下
    • 五环以内贷款上限304万 368w*65% = 303w
    • 五环以外贷款上限243万 374w*65% = 243w
      首付款-税费+可贷款额度=购房房屋价格
  • 普通住宅契税政策
    90平米以下1%
    90平米-144平米以上1.5%
    144平米以上3%
    非普通住宅，二套及以上3%

选房 中介带看房

  • 预算
    • 区域
    • 交通
    • 房龄
    • 房型
    • 面积
    • 朝向
    • 楼层
    • 物业
    • 绿化
    • 是否满五唯一
    • 是否可以高贷款
    1. 高层住宅3层以下不买
    2. 距地铁步行不要超过20分钟
    3. 户型方正
    4. 物业负责
      满五唯一：指该省份内所在城市，你就只有1套房子，且这套房子产证已经满5年了就是满5年唯一。是以家庭为单位的，在该省份内。通常情况下，房子满足“满五唯一”的条件即可减免房子的个税和营业税。
      只需交1%的契税。不唯一交20%个人所得税。
  • 房屋类型
    • 不可转普商
      商住两用类
      商业办公类
    • 可转商
      经济适用房
      限价商品房
      已购公房
      央产房

面谈价格

  • 会议室 双方会谈 中介传话
  • 直接面对业主

签约

  • 核验房产证
  • 权属信息 （双方都要签字）
  • 有无出租 （买卖不破租赁）
  • 有无户口 （户口迁出时间）
  • 腾房时间
  • 屋内设施归属
  • 有无占用学位 （6月份上学）
  • 定金支付时间
  • 首付支付时间
  • 户口保证金 （过户后付款，业主户口迁出，物业交割完成）
  • 物业保证金 （过户后付款，业主户口迁出，物业交割完成）
  • 是否需要解抵押 （首付解押）
    违约金 天/万分之一
    北京一般直接签正式合同，违约金高，不是意向合同

资质审核

  • 商贷 5.2%
  • 公积金贷款 3.25%
    组合贷：北京市管公积金贷款按公积金缴存年限计算，一年10w 最高120w
    国管公积金，公积金余额12倍，最高120w
    贷款注意年收入要高于年还款额的2倍。
    北京最高贷款 25年 （7 8 年换房 等额本息）
